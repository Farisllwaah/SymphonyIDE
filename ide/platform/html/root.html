Comprehensive Modelling for Advanced Systems of Systems



The COMPASS Tool

The COMPASS tool aims to support the building of models of SoSs and the analysis of global properties on these modeles.

Getting Started

  • The user manual (D31_2a) can be found in the SVN repository in Theme3/WP31.
  • CML examples can also be found in the project repository under Common/CaseStudies.

Submitting bugs

The COMPASS tool is still in its early stages of development. You can help improve it by submitting bugs or requesting new features

  • The COMPASS bug tracker is available here.
  • You can also request features requests here.
  • Please note that a Sourceforge account is necessary to submit feature requests.
For general information about COMPASS see compass-research.eu.
